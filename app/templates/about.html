What's this?

This is a Home Server that allows users connected to the same WiFi network to share pictures, documents and other files. Written with Flask and Bootstrap. More info.

This project started when I've decided to create a file sharing platform that would double as a local social media network, where users are able to follow, message and share files with each other. Although far from perfect, this is a great introductory to Flask and Bootstrap web development. If you're interested, the project is hosted here.
